How to play:

The game consists of 5 rounds. Each time you or the computer scores a point. If there is a tie, no one gets a point. In the world of magical cats, there are still certain rules: the wizard defeats the assassin, the fairy defeats the wizard, and the assassin defeats the fairy. Two cats of the same class cannot defeat each other. It is very easy. Have a nice game! ♡
